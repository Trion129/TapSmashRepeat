# Game entry for Minimalistic GameJam 3 on itch.io

## About

It is game where you try to tap to break a block before the opponent.
There are 3 modes:

1. Wood - 30 taps
2. Stone - 70 taps
3. Obsidian - 130 taps

The person that breaks first wins.

## Technologies used

NodeJS
Photoshop
Phaser
